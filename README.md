### Q&A

> 关于XLIM增强型协议？

XLIM增强型协议极其客户端实现的复杂性是普通LIM协议的数倍，其提供了更易于使用的操作模式（甚至图形化界面），以及更完善的通信流程。

**（XLIM协议更新频率较慢，其协议规范与LIM分离）**
